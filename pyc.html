سرفصل‌های دوره آموزشی پایتون: از مقدماتی تا پیشرفته

این سند سرفصل‌های جامع دوره آموزشی پایتون را ارائه می‌دهد که برای یادگیری کامل و حرفه‌ای زبان برنامه‌نویسی پایتون طراحی شده است. این دوره تمامی مفاهیم از سطح مقدماتی تا پیشرفته را پوشش می‌دهد و با آخرین نسخه پایتون (3.13.5) هم‌راستا است.

ماژول 1: مقدمه‌ای بر پایتون

  • پایتون چیست؟: معرفی زبان برنامه‌نویسی پایتون و کاربردهای آن در حوزه‌های مختلف.
  • تاریخچه و تکامل پایتون: مروری بر سیر تحول پایتون از نسخه‌های اولیه تا 3.13.
  • چرا پایتون؟: مزایای پایتون مانند خوانایی، سادگی، و کتابخانه‌های گسترده.
  • نصب و راه‌اندازی محیط پایتون: نصب پایتون، انتخاب IDE (مانند PyCharm، VSCode)، و تنظیم محیط توسعه.
  • نوشتن و اجرای اولین برنامه: نوشتن برنامه ساده "Hello, World!" و اجرای آن.
  • ساختار و سینتکس پایتون: آشنایی با قوانین پایه سینتکس پایتون.

نکات کلیدی:

  • پایتون یک زبان چندمنظوره است که در توسعه وب، علم داده، اتوماسیون، و هوش مصنوعی کاربرد دارد.
  • استفاده از محیط‌های مجازی (Virtual Environments) برای مدیریت وابستگی‌ها توصیه می‌شود.
موضوع توضیحات منابع پیشنهادی
نصب پایتون راهنمای نصب پایتون 3.13 در ویندوز، مک، و لینوکس python.org/downloads
انتخاب IDE مقایسه PyCharm، VSCode، و Jupyter Notebook realpython.com

ماژول 2: مفاهیم پایه پایتون

  • متغیرها و انواع داده: اعداد (int, float)، رشته‌ها (str)، بولین‌ها (bool).
  • عملگرها: حسابی (+, -, *, /)، مقایسه‌ای (==, !=, >)، منطقی (and, or, not)، و تخصیص (=).
  • ورودی و خروجی: استفاده از توابع print() و input().
  • توضیحات (Comments): نوشتن توضیحات تک‌خطی و چندخطی.
  • تبدیل نوع (Type Casting): تبدیل بین انواع داده مانند int()، str()، و float().

نکات کلیدی:

  • پایتون به‌صورت پویا نوع داده‌ها را تعیین می‌کند (Dynamic Typing).
  • از نام‌گذاری معنادار برای متغیرها استفاده کنید (مثلاً user_age به‌جای x).
عملگر مثال توضیح
حسابی a + b جمع دو عدد
مقایسه‌ای a == b بررسی برابری
منطقی a and b بررسی شرط‌های منطقی

ماژول 3: ساختارهای کنترلی

  • دستورات شرطی: if, elif, else برای تصمیم‌گیری.
  • حلقه‌ها: حلقه‌های for و while برای تکرار.
  • کنترل حلقه‌ها: استفاده از break, continue, pass.
  • حلقه‌ها و شرط‌های تو در تو: ترکیب شرط‌ها و حلقه‌ها برای منطق پیچیده‌تر.

نکات کلیدی:

  • از تورفتگی (Indentation) صحیح برای تعریف بلوک‌های کد استفاده کنید.
  • حلقه‌های تو در تو می‌توانند عملکرد را کاهش دهند؛ از آن‌ها با دقت استفاده کنید.
ساختار مثال کاربرد
if if x > 0: بررسی شرط
for for i in range(5): تکرار در یک محدوده
while while x < 10: تکرار تا زمانی که شرط برقرار است

ماژول 4: توابع

  • تعریف و فراخوانی توابع: استفاده از کلمه کلیدی def.
  • پارامترهای تابع: موقعیتی، کلیدواژه‌ای، پیش‌فرض، و متغیر (*args, **kwargs).
  • مقدار بازگشتی: استفاده از return برای بازگشت مقادیر.
  • دامنه و طول عمر متغیرها: متغیرهای محلی و جهانی.
  • توابع لامبدا: توابع بی‌نام برای عملیات کوتاه.
  • بازگشتی (Recursion): حل مسائل با فراخوانی تابع درون خودش.
  • دکوراتورها: افزودن قابلیت به توابع با استفاده از @.

نکات کلیدی:

  • توابع باید تک‌منظوره و خوانا باشند.
  • دکوراتورها برای افزودن رفتارهای اضافی به توابع بدون تغییر کد آن‌ها مفیدند.
نوع تابع مثال توضیح
معمولی def add(a, b): return a + b تابع ساده
لامبدا lambda x: x * 2 تابع بی‌نام
دکوراتور @my_decorator افزودن رفتار به تابع

ماژول 5: ساختارهای داده

  • لیست‌ها: ایجاد، دسترسی، برش (Slicing)، متدها، و List Comprehension.
  • تاپل‌ها: ایجاد، تغییرناپذیری، و متدها.
  • دیکشنری‌ها: جفت‌های کلید-مقدار، متدها، و Dictionary Comprehension.
  • مجموعه‌ها (Sets): ایجاد، عملیات مجموعه‌ای (اتحاد، اشتراک)، و متدها.
  • رشته‌ها: دستکاری، متدها، و قالب‌بندی رشته‌ها (f-strings).

نکات کلیدی:

  • لیست‌ها قابل تغییر هستند، اما تاپل‌ها خیر.
  • از f-strings برای قالب‌بندی رشته‌ها به دلیل خوانایی بالا استفاده کنید.
ساختار داده مثال ویژگی‌ها
لیست [1, 2, 3] قابل تغییر، مرتب
تاپل (1, 2, 3) غیرقابل تغییر، مرتب
دیکشنری {"name": "Ali"} کلید-مقدار، غیرمرتب
مجموعه {1, 2, 3} بدون تکرار، غیرمرتب

ماژول 6: ماژول‌ها و بسته‌ها

  • وارد کردن ماژول‌ها: استفاده از import برای دسترسی به ماژول‌ها.
  • ماژول‌های کتابخانه استاندارد: مانند math, random, datetime.
  • ایجاد و استفاده از بسته‌ها: سازمان‌دهی کد در بسته‌های پایتون.
  • مدیریت کتابخانه‌های شخص ثالث: نصب با pip.

نکات کلیدی:

  • ماژول‌ها کد را ماژولار و قابل استفاده مجدد می‌کنند.
  • از pip برای نصب کتابخانه‌هایی مانند requests یا numpy استفاده کنید.
ماژول کاربرد مثال
math عملیات ریاضی math.sqrt(16)
random تولید اعداد تصادفی random.randint(1, 10)
datetime کار با تاریخ و زمان datetime.now()

ماژول 7: مدیریت فایل‌ها

  • خواندن و نوشتن فایل‌ها: کار با فایل‌های متنی.
  • حالت‌های فایل: خواندن (r)، نوشتن (w)، افزودن (a).
  • کار با فرمت‌های مختلف: فایل‌های CSV، JSON.
  • مدیریتگرهای زمینه (Context Managers): استفاده از with برای مدیریت منابع.

نکات کلیدی:

  • استفاده از with از نشت منابع (Resource Leak) جلوگیری می‌کند.
  • برای فایل‌های JSON از ماژول json استفاده کنید.
عملیات مثال توضیح
خواندن with open('file.txt', 'r') as f: خواندن محتوای فایل
نوشتن with open('file.txt', 'w') as f: نوشتن در فایل

ماژول 8: مدیریت استثناها

  • درک استثناها و خطاها: تفاوت بین خطاهای سینتکسی و استثناها.
  • بلوک‌های Try-Except: مدیریت خطاها.
  • مدیریت استثناهای خاص: مانند ValueError, FileNotFoundError.
  • پرتاب استثناها: استفاده از raise.
  • بند Finally: اجرای کد پس از مدیریت استثنا.
  • استثناهای سفارشی: تعریف استثناهای کاربر.

نکات کلیدی:

  • استثناهای خاص را مدیریت کنید تا کد قوی‌تر شود.
  • از finally برای آزادسازی منابع استفاده کنید.
استثنا مثال توضیح
ValueError int("abc") خطای تبدیل نوع
FileNotFoundError open("missing.txt") فایل یافت نشد

ماژول 9: برنامه‌نویسی شیءگرا (OOP)

  • کلاس‌ها و اشیاء: تعریف و استفاده.
  • ویژگی‌ها و متدها: متدهای نمونه، کلاسی، و استاتیک.
  • سازنده‌ها و تخریب‌کننده‌ها: __init__ و __del__.
  • وراثت: وراثت تک، چندگانه، و چندسطحی.
  • چندریختی (Polymorphism): بازنویسی متدها و بارگذاری عملگرها.
  • کپسوله‌سازی: اعضای خصوصی و محافظت‌شده.
  • متدهای ویژه (Dunder Methods): مانند __str__, __len__.

نکات کلیدی:

  • OOP کد را ماژولار و قابل نگهداری می‌کند.
  • از متدهای ویژه برای سفارشی‌سازی رفتار اشیاء استفاده کنید.
مفهوم مثال توضیح
کلاس class Dog: تعریف یک کلاس
وراثت class Puppy(Dog): ارث‌بری از کلاس پایه

ماژول 10: موضوعات پیشرفته

  • تکرارکننده‌ها و مولدها (Iterators & Generators): ایجاد و استفاده.
  • دکوراتورها: دکوراتورهای تابع و کلاس.
  • مدیریتگرهای زمینه: ایجاد مدیریتگرهای سفارشی.
  • متاکلاس‌ها: کنترل ایجاد کلاس‌ها.
  • توصیف‌گرها (Descriptors): مدیریت دسترسی به ویژگی‌ها.
  • برنامه‌نویسی تابعی: استفاده از map, filter, reduce.
  • عبارات منظم (Regular Expressions): کار با ماژول re.

نکات کلیدی:

  • مولدها حافظه را بهینه می‌کنند.
  • عبارات منظم برای پردازش متن پیچیده مفیدند.
موضوع مثال توضیح
مولد def my_gen(): yield 1 تولید مقادیر به‌صورت تنبل
عبارت منظم re.match(r'\d+', '123') تطبیق الگوها

ماژول 11: همزمانی و موازی‌سازی

  • رشته‌ها (Threading): ایجاد و مدیریت رشته‌ها.
  • چندپردازشی (Multiprocessing): اجرای فرآیندهای موازی.
  • برنامه‌نویسی ناهمگام (Asyncio): استفاده از کوروتین‌ها.
  • همگام‌سازی: قفل‌ها، سمافورها، و سایر مکانیزم‌ها.

نکات کلیدی:

  • برای وظایف I/O-bound از asyncio و برای CPU-bound از multiprocessing استفاده کنید.
  • همگام‌سازی از مشکلات دسترسی همزمان جلوگیری می‌کند.
روش مثال کاربرد
Threading threading.Thread(target=func) اجرای همزمان وظایف سبک
Asyncio async def my_func(): مدیریت عملیات ناهمگام

نکات کلیدی:

  • Scikit-learn یادگیری ماشین را ساده می‌کند.

NumPy عملیات آرایه‌ای np.array([1, 2, 3])
Pandas تحلیل داده pd.DataFrame(data)
